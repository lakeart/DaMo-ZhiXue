## 5. 代码核查纠偏说明（完整争议回应清单）

以下为针对常见评审质疑的逐项回应：

1. ✅ **RAG检索层**：默认TF-IDF(基于scikit-learn)，已补充ChromaDB可选向量检索；ChromaDB不可用时自动回退TF-IDF
2. ✅ **RAG回答层**：完整链路为"检索证据+LLM生成+引用保留+后处理(Mock检测/幻觉检测)+兜底模板"，非简单规则拼接
3. ✅ **知识库上传**：支持8种格式(.txt/.md/.csv/.json/.docx/.pdf/.pptx/.ppt)，`.pptx`稳定解析，`.ppt`兼容提取
4. ✅ **学习路径**：甘特图、日历、列表三视图在当前模板中存在并可切换，非缺失项
5. ✅ **核心文档**：SRS/开发/测试/用户/创新/开源/路径/清单 — 8份文档全部v3.0深度完善
6. ✅ **课程样例包**：6个真实课程文件 + 真实上传/索引/检索/RAG问答全链路验证（JSON证据在`docs/evidence/api/`）
7. ✅ **多Agent协同**：7个Agent独立可用 + Coordinator统一调度 + 执行日志可追溯
8. ✅ **安全机制**：RAG引用+内容安全过滤+幻觉检测+Mock检测+资料不足拦截 — 多重保障
